Adeept Pi Car

Un robot intelligent pour apprendre l'IA, la robotique et l'électronique

Acheter

Voiture intelligente

Adeept Kit Voiture Intelligente PiCar-B Mars Rover pour Raspberry Pi

  • Comprend un kit de voiture de robot AI basé sur Raspberry Pi
  • Conçu pour enseigner l'IA, la robotique et l'électronique
  • Offre le suivi des objets avec OpenCV
  • Il est conçu pour les débutants et les professionnels pour apprendre l'IA, la robotique et l'électronique
  • Ce kit de robot éducatif STEM est basé sur le Raspberry Pi (compatible avec RPi 3B / 3B + / 2B / 2B +, le Raspberry Pi n'est PAS inclus).
  • Le PiCar-B peut être contrôlé par la voix
  • Il offre la reconnaissance et le suivi des objets, basés sur OpenCV
  • Dispose d'une fonction d'évitement automatique des obstacles, basée sur le capteur à ultrasons
  • L'utilisation de la réflexion infrarouge permet d'effectuer le suivi de ligne.
